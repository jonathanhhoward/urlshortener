URL Shortener Microservice

Short URL Creation

Example Usage

POST [server_url]/api/shorturl/new -- www.your-url.com
Returns: {"original_url":"www.your-url.com","short_url":"your-id-string"}

GET [server_url]/api/shorturl/your-id-string
Redirects To: https://www.your-url.com

Bad URL Returns: {"error":"invalid URL"}
